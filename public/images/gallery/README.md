# Gallery Images / صور المعرض

## هيكل المجلد / Folder Structure

قم برفع صور أعمالك هنا بالتنسيق التالي:

```
gallery/
├── projekt-1-vorher.jpg    (صورة قبل - المشروع 1)
├── projekt-1-nachher.jpg   (صورة بعد - المشروع 1)
├── projekt-2-vorher.jpg    (صورة قبل - المشروع 2)
├── projekt-2-nachher.jpg   (صورة بعد - المشروع 2)
├── projekt-3-vorher.jpg    (صورة قبل - المشروع 3)
├── projekt-3-nachher.jpg   (صورة بعد - المشروع 3)
└── ...
```

## متطلبات الصور / Image Requirements

- **الحجم المُوصى به / Recommended Size:** 800x600 pixels
- **التنسيقات المدعومة / Supported Formats:** JPG, PNG, WebP
- **حجم الملف الأقصى / Max File Size:** 500KB (للتحميل السريع)

## تسمية الملفات / File Naming

- استخدم أسماء واضحة مثل: `wohnung-nuernberg-vorher.jpg`
- تجنب المسافات والأحرف الخاصة
- استخدم `-vorher` لصور "قبل" و `-nachher` لصور "بعد"

## مثال / Example

```
wohnung-nuernberg-vorher.jpg
wohnung-nuernberg-nachher.jpg
haus-fuerth-vorher.jpg
haus-fuerth-nachher.jpg
keller-erlangen-vorher.jpg
keller-erlangen-nachher.jpg
```
